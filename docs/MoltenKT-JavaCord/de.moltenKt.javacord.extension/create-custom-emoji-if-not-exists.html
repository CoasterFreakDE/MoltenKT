Creates a custom emoji using the CustomEmojiBuilder and returns the created emoji. If an emoji with the name emojiName already exists, the existing emoji will be returned and no new emoji will be created and stored on the server.

Return

the created or already existing emoji

emojiName

the (new) name of the emoji

resource

the image file of the emoji (png) as a byte array

process

a custom function which can be used to modify the CustomEmojiBuilder after all parameters set into it.
