suspend fun <T> asSync(process: Tasky.() -> T): T
Content copied to clipboard

This function creates a sync task using the Tasky system. Internally a CompletableFuture is used to create the delayed result. Because the CompletableFuture.await function uses the coroutine suspend feature, this function also utilizes the suspend function feature, so the result can be easily managed into your existing MoltenKT App.
