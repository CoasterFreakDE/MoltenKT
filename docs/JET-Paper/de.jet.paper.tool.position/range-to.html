operator fun Location.rangeTo(other: Location): LocationBox

This function creates a new LocationBox object, representing a range from one to another Location. This function is defined as a rangeTo operator function. This function uses the this as the first Location (LocationBox.component1) and the other (LocationBox.second) as the second Location inside the LocationBox.

Return

a new LocationBox object.
